# 自媒体工作室管理工具

一个专为自媒体工作室设计的纯前端管理工具，支持人员管理、项目管理、任务管理和AI提示词模板。可部署到GitHub Pages并作为PWA添加到iPhone主屏幕使用。

## 功能模块

### 1. 人员管理
- 记录团队成员信息
- 设置内容类型（真人、配音、剪辑、实拍、直播、其他）
- 记录价格和产能
- 自动统计每日总产能

### 2. 项目管理
- 记录甲方签约项目
- 内容报价（支持多类型，单价×数量计算总价）
- 结算方式（内容发布账号/任意账号/对公结算/其他）
- 结算标准（推荐/优质/人工）
- 形式要求、内容规格、质量要求
- 账号数量相关设置
- 活动链接
- **智能识别**：粘贴合作邀约文字自动填充字段
- **导出功能**：支持导出文字（键值对）或图片

### 3. 任务管理
- 关联项目自动同步信息
- 执行人员分配
- 时间设置
- 内容规格（形式要求、内容长度、视频格式、封面要求）
- 质量要求
- **通用质量要求**（多选）：
  - 素材不能出现水印、品牌logo、真人人脸等敏感信息
  - 文案与画面完全对应
  - 封面图必须包含清晰、完整的车
  - 文案有真人感，禁止AI味
  - 需要植入关键词
  - 其他（可自定义）
- 价格设置
- **导出功能**：支持导出文字或图片

### 4. 提示词工具（5个模板）

#### 模板1：视频标题+指定信息源
根据指定网站搜索生成短视频标题

#### 模板2：赛事视频标题
搜索赛事信息生成吸睛标题

#### 模板3：赛事视频文案
生成自然口播文案，支持风格参考

#### 模板4：线路标题改写
跑山/越野线路标题批量改写

#### 模板5：玩车系标题
汽车主题吸睛标题生成

**提示词功能特点**：
- 可视化预览，蓝色高亮挖空位置
- 表单输入与预览双向同步
- 一键复制完整提示词
- 关闭后自动重置默认值

## 技术栈

- HTML5 + CSS3 + JavaScript (ES6+)
- LocalStorage 数据持久化
- PWA (Service Worker + Manifest)
- 响应式设计，支持移动端和桌面端
- 深色主题，iOS风格界面

## 部署到 GitHub Pages

### 1. 准备文件
项目文件结构：
```
studio-manager/
├── index.html          # 主页面
├── manifest.json       # PWA配置
├── sw.js              # Service Worker
├── icon.png           # 应用图标（已提供）
├── css/
│   └── style.css      # 样式文件
├── js/
│   ├── data.js        # 数据管理
│   ├── projects.js    # 项目管理
│   ├── tasks.js       # 任务管理
│   ├── people.js      # 人员管理
│   ├── tools.js       # 提示词工具
│   └── app.js         # 主应用
└── README.md          # 说明文档
```

### 2. 创建 GitHub 仓库
1. 登录 GitHub，点击右上角 "+" → "New repository"
2. 仓库名称：`studio-manager`（或其他名称）
3. 选择 "Public"（公开）
4. 点击 "Create repository"

### 3. 上传文件
**方式一：直接上传（推荐）**
1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 拖拽或选择所有项目文件上传
3. 点击 "Commit changes"

**方式二：Git命令**
```bash
cd studio-manager
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/studio-manager.git
git push -u origin main
```

### 4. 启用 GitHub Pages
1. 进入仓库的 "Settings" 标签
2. 左侧菜单点击 "Pages"
3. "Source" 选择 "Deploy from a branch"
4. "Branch" 选择 "main"，文件夹选择 "/ (root)"
5. 点击 "Save"
6. 等待几分钟，页面会显示访问链接：`https://你的用户名.github.io/studio-manager/`

## iPhone PWA 安装

### 添加到主屏幕
1. 用 Safari 打开部署后的链接
2. 点击底部分享按钮（方框带向上箭头）
3. 选择 "添加到主屏幕"
4. 点击 "添加"
5. 即可像原生App一样使用

### PWA 特性
- 全屏显示，无浏览器地址栏
- 离线可用（数据存储在本地）
- 深色主题适配iOS
- 流畅的底部导航

## 数据备份

由于数据存储在浏览器 LocalStorage 中：
- **定期导出**：建议定期导出数据备份
- **跨设备同步**：可通过导出/导入JSON文件在不同设备间同步
- **清除风险**：清除浏览器数据会导致数据丢失

## 浏览器兼容性

- Safari (iOS 14+) - 推荐，PWA支持最佳
- Chrome/Edge (桌面和移动端)
- 其他现代浏览器

## 注意事项

1. **首次加载**：PWA首次加载需要联网，之后可离线使用
2. **数据同步**：如需多设备同步，请手动导出导入JSON文件
3. **图标显示**：确保 `icon.png` 文件存在，用于iPhone主屏幕图标
4. **GitHub Pages**：免费版有流量限制，适合个人/小团队使用
